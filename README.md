# ToDoListApp

Android mobile application that show all tasks and can add task and change task to be completed or not and delete task or delete all tasks, 

# Technologies & Libraries:

- Kotlin.
- MVVM.
- Clean Architecture.
- RoomDataBase.
- Coroutine.
- Kotlin Flows.
- ViewBinding
- Navigation Component
- Hilt
